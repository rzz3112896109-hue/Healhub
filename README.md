# HealHub 智慧社区疗愈平台

一个基于 React、TailwindCSS 与 React Router 的响应式疗愈社区网页项目。

## 直接打开

双击项目根目录的 `index.html` 即可直接使用，不需要安装软件或启动本地服务。该文件已内嵌完整的 React、样式、图表与页面路由。

## 功能

- 首页：今日情绪打卡、AI 疗愈入口、社区动态与附近活动
- 情绪空间：情绪选择、本地存储、日历与趋势图表
- AI 疗愈室：模拟对话、冥想计时器与呼吸训练
- 社区疗愈圈：匿名发帖、点赞与评论支持
- 线下活动：活动筛选、报名与取消报名
- 个人中心：疗愈统计、AI 月度报告与成长徽章

## 本地运行

```bash
npm install
npm run dev
```

生产构建：

```bash
npm run build
```

所有体验数据仅存储在当前浏览器的 `localStorage` 中，不会上传到服务器。
